# Link Health Check

## Purpose

Keep the Digital Support Index trustworthy.

## When

Use during MAINT, WEEK-GOV, or support index maintenance.

## Check

- Do route card links exist?
- Do governance contract links exist?
- Do state source links exist?
- Do evidence target links exist?
- Are external URLs still valid?
- Are placeholders still acceptable?
- Are there stale routes that no longer appear in the governance map?

## Output

One of:

- links OK;
- missing links added;
- stale links removed;
- placeholder marked;
- route parked;
- follow-up created.

## Stop rule

Do not redesign the whole system during link checking. Fix links or create next pointers.
