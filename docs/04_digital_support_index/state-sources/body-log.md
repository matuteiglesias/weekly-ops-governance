# Body Log

## Role

Source of truth for body signals such as sleep, weight, measures, alcohol, symptoms, and recovery constraints.

## Used by

- BODY-GOV
- GYM-GOV
- WEEK-GOV if body constraints affect planning

## Expected fields

- Date
- Sleep/wake
- Weight/measure, if recorded
- Alcohol signal
- Symptoms/fatigue
- Recovery note
- Follow-up if needed
