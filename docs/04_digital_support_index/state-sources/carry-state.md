# Carry State

## Role

Source of truth for active fronts, project status, blockers, parked reasons, and next pointers.

## Used by

- CARRY-M
- CARRY-W
- WEEK-GOV
- FOCUS
- ACTIVE-FRONTS
- PROJECT-TRIAGE

## Expected fields

| Field | Meaning |
|---|---|
| Front | Project/front name. |
| Status | active, paused, waiting, done, not-now. |
| Next pointer | Small restart instruction. |
| Evidence | Link to recent output. |
| Blocker | Current obstacle if any. |
| Cadence | Expected next touch. |
| Owner/mode | Optional role or mode. |
