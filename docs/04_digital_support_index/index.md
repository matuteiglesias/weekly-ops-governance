# Digital Support Index

The Digital Support Index is the navigation layer for the weekly operations governance system.

It is not a weekly board and does not use time/day geometry.

It maps:

- route ID;
- domain;
- route card;
- governance contract;
- state source;
- checklist/manual;
- evidence source;
- cadence;
- next action.

## Purpose

When the weekly board shows an ID such as `FIN-W`, `CARRY-W`, `GYM-GOV`, or `WEEK-GOV`, this index helps the operator reach the correct supporting material without searching.
