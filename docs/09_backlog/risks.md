# Risks

| Risk | Symptom | Response |
|---|---|---|
| Board overload | Weekly page becomes dense and avoided | Move content to route cards/support index |
| Link drift | Route IDs point to stale or missing files | Run link health check |
| Meta-work expansion | Operating system becomes the work | Use printables for one week before expanding |
| Fake completion | Ticks happen but no state/evidence changes | Check output/evidence section |
| Monday restart friction | Monday requires reconstructing context | Use separate Monday Bridge file |
| Capture creep | Free notes flood weekly board | Route through CAPTURE-GOV only |
| Gym drift | Gym block becomes improvised | Use GYM-GOV contract |
| Finance drift | Vencimientos and ledger become ambiguous | Use FIN-W Friday + Sunday/Monday |
