# Print Style Guide

## Priorities

1. Readability.
2. Space for handwriting.
3. Clear checkboxes.
4. Low visual density.
5. Black-and-white friendly.
6. No dependence on color.

## Page defaults

- A4
- Portrait for guide/cards
- Landscape acceptable for weekly board later
- Generous margins
- Avoid tiny tables
- Avoid dense paragraphs on working pages

## Checkbox convention

- `□` means actionable tick.
- Blank lines mean handwriting space.
- Route IDs should stay uppercase and stable.

## Rule

If a page cannot be used with a pen, it is not a working printable.
