# Printables

Printable artifacts are the paper-facing outputs of the system.

## Start here

- Use the [Printables Manifest](printables-manifest.md) to see source files and generated site assets.
- Print the [Weekly Board PDF](/printables/weekly-board-v1.pdf) as the main weekly surface.
- Print the [Monday Bridge PDF](/printables/monday-bridge-v1.pdf) as the separate week-restart sheet.
