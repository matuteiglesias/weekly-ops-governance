# NEXT-GOV — Continuity Contract

## Purpose

Make tomorrow or next week restartable without reconstructing context.

## Contract

A next pointer is valid when it names:

- the next action;
- the artifact or context;
- the first step;
- any warning or boundary.

## Inputs

- Current block result
- Artifact link/path
- Open blocker
- Board annotation
- Carry State if project-level

## Outputs

- Next pointer
- Restart note
- Parked reason
- Monday bridge

## Tick valid when

A future block can restart without reconstructing the thread.

## Stop rule

A next pointer is not a report. It should be short enough to use under low energy.
