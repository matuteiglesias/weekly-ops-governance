# WEEK-GOV — Weekly Governance

## When

Use Sunday bridge, Monday preparation, or a scheduled weekly anchor.

## Purpose

Orient the week without overloading the weekly board.

## Inputs

- Weekly board
- Previous weekly packet
- Carry State
- Active fronts
- Outputs from last week
- Known blockers
- 14-day or quarterly context if active

## Steps minimum

1. Review last week's visible outputs.
2. Check active fronts.
3. Choose the week shape.
4. Name 1-3 outputs.
5. Identify blockers and not-now items.
6. Confirm finance/carry/house/body/gym routes that matter.
7. Leave a Monday bridge if needed.

## Output

- Week orientation
- Active fronts
- Output list
- Blockers/not-now list
- Monday bridge

## Tick valid when

The week has a readable shape and the next work block can be selected.

## Stop rule

Do not convert weekly governance into a full life strategy session unless explicitly scheduled.
