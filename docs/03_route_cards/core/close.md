# CLOSE — Daily Close

## When

Use at the end of the workday or before sleep when the day touched active work.

## Purpose

Prevent context loss.

## Inputs

- Today's board marks
- Work artifacts
- Open notes
- Block results
- Carry State if needed

## Steps minimum

1. Identify what changed today.
2. Write the smallest next pointer.
3. Mark unresolved blocker if relevant.
4. Park loose notes or route them to capture.
5. Leave tomorrow cheaper to start.

## Output

A short close note or next pointer.

## Tick valid when

Tomorrow's first restart point is clear enough.

## Stop rule

Do not run a full weekly review during CLOSE. Keep it small.
