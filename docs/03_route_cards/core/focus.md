# FOCUS — Deep Work Block

## When

Use for 9-12, 13-16, or 17-20 focus blocks.

## Purpose

Convert focused time into evidence.

## Inputs

- Selected front
- Relevant artifact/repo/doc/sheet
- Expected output
- Context from BOOT or Carry State
- Stop rule

## Steps minimum

1. Open only the artifact needed for the block.
2. Re-state the output.
3. Work toward the output.
4. Capture major decisions only if needed.
5. Leave evidence or next pointer.

## Output examples

- Draft completed
- Section rewritten
- Message sent
- Commit made
- Report generated
- Table updated
- Design decision captured
- Demo deployed
- Next pointer written

## Tick valid when

The block leaves an observable artifact, state update, or useful next pointer.

## Stop rule

Do not turn focus into system redesign unless the selected focus output is explicitly a design artifact.
