# TRAVEL-GOV — Travel Admin Governance

## When

Use only when a travel front is active.

## Purpose

Keep travel documents, logistics, receipts, and admin tasks organized without making travel a permanent weekly burden.

## Inputs

- Travel checklist
- Passport/visa docs
- Itinerary
- Lodging
- Receipts
- Budget/logistics notes

## Steps minimum

1. Identify active trip/admin need.
2. Check required document/action.
3. Complete or park one item.
4. Update travel state.
5. Stop.

## Output

Travel admin item resolved or clearly parked.

## Tick valid when

One concrete travel/logistics item has a known status.

## Stop rule

Do not include Travel-GOV on the weekly board unless travel is active.
