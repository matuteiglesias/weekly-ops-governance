# ACADEMIC-GOV — Academic Work Governance

## When

Use when thesis, research, corpus, supervision, or academic positioning is active.

## Purpose

Convert academic work into presentable artifacts instead of endless accumulation.

## Inputs

- Thesis/project docs
- Literature corpus
- Advisor/supervision context
- Drafts
- Academic CV/materials
- Current deadline

## Steps minimum

1. Choose the academic artifact.
2. Define expected output.
3. Work toward presentability.
4. Record citation/source needs if any.
5. Leave next pointer.

## Output

Academic artifact advanced.

## Tick valid when

A draft, figure, section, bibliography item, abstract, or supervision artifact becomes more usable.

## Stop rule

Do not expand reading unless reading is the selected output.
