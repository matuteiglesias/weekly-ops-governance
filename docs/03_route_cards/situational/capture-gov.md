# CAPTURE-GOV — Capture Governance

## When

Use inside WEEK-GOV, capture processing, or review sessions.

## Purpose

Route free notes without letting them pollute the weekly board.

## Inputs

- Free notes
- Capture inbox
- Capture manual
- Current active fronts
- Week review context

## Steps minimum

1. Gather loose notes.
2. Route each note to action, state, archive, waiting, or discard.
3. Create tasks only when justified.
4. Update state if the note changes a project.
5. Stop when the capture surface is no longer ambiguous.

## Output

Loose notes routed.

## Tick valid when

Important notes are no longer floating.

## Stop rule

Do not process capture directly from the weekly board unless explicitly scheduled.
