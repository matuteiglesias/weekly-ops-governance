# BREAKTHROUGH-14D — Two-Week Breakthrough Frame

## When

Use when a seasonal or quarterly direction needs a concrete two-week execution frame.

## Purpose

Convert a broad ambition into a 14-day visible change of state.

## Inputs

- Seasonal frame
- Active fronts
- Governance map
- Current bottleneck
- External/user-facing target

## Steps minimum

1. Choose one breakthrough.
2. Define current state.
3. Define target state after 14 days.
4. Name 2-3 visible outputs.
5. Choose review point at day 7.
6. Write what is explicitly not included.

## Output

A 14-day frame with outputs and boundaries.

## Tick valid when

The frame can guide actual weekly board decisions.

## Stop rule

Do not define more than one primary breakthrough per 14-day cycle.
