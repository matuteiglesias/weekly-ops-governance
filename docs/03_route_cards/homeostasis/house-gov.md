
# HOUSE-GOV — House Operations Governance

## When

Use during MAINT slots, Saturday homeostasis, or whenever house friction is visibly affecting execution.

## Purpose

Prevent the physical environment from becoming hidden operational debt.

## Inputs

* House checklist
* Cleaning route
* Laundry state
* Kitchen/bathroom observation
* Local friction

## Steps minimum

1. Choose one bounded house target.
2. Set a 20-30 minute cap.
3. Improve one real zone or routine.
4. Mark done or next house pointer.
5. Stop when the cap ends.

## Output

* Improved zone;
* trash/laundry/kitchen/bathroom action;
* visible reset;
* next house item if needed.

## Tick valid when

A real physical zone or routine improved within the bounded session.

## Stop rule

Do not turn CLN-30 into a full-house perfection run.
