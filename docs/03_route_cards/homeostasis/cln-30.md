# CLN-30 — Bounded Cleaning Session

## When

Use during MAINT, Saturday homeostasis, or when house friction is visible.

## Purpose

Improve the physical environment with a bounded 20-30 minute action.

## Inputs

- House checklist
- Current room/zone observation
- Trash/laundry/kitchen/bathroom state
- Timer

## Steps minimum

1. Pick one zone or routine.
2. Set a 20-30 minute timer.
3. Do visible improvement.
4. Stop at the cap.
5. Mark done or next house pointer.

## Good targets

- Trash
- Dishes/kitchen
- Laundry
- Bathroom reset
- Desk/surface
- Floor sweep
- Clothing pile
- Bed/sheets

## Output

One improved zone or routine.

## Tick valid when

A real physical improvement happened inside the bounded session.

## Stop rule

Do not expand into full-house reset.
